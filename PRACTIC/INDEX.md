## 📞 Support & Troubleshooting

### Common Issues & Solutions

**All services not starting**
```bash
docker-compose config          # Validate compose file
docker-compose logs            # Check error logs
docker-compose up --build      # Rebuild all images
```

**Port conflicts**
```bash
lsof -i :80                    # Find process using port
kill -9 <PID>                  # Kill the process
```

**Database connectivity issues**
```bash
docker-compose exec postgres psql -U demo -d demo -c "SELECT 1;"
docker-compose logs postgres   # Check PostgreSQL logs
docker-compose restart postgres
```

**Rate limiting not working**
```bash
docker-compose logs rate-limiter
curl -v http://localhost:3000/health
```
